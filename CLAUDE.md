## 프로젝트 개요

**OmniMind(옴니마인드)** 는 사주팔자·혈액형·MBTI·별자리 등 파편화된 개인 데이터를 연결해 사용자가 '온전한 나'를 발견하도록 돕는 **모바일 웹 기반 서비스**입니다. 이름은 **Omni(모든 것) + Mind(마음)** 의 결합.

- **한 줄 정의:** 단순 운세가 아닌 "생애 주기별 멘탈 가이드"
- **저장소:** https://github.com/swssmjy86/OmniMind.git

## 서비스 구조

### 기본 입력 (온보딩)

| 입력 | 용도 |
|------|------|
| 생년월일시 | 사주팔자 (만세력 기반) |
| 혈액형 | 혈액형 4가지 성격 유형 |
| MBTI | MBTI 16가지 성격 유형 |
| 별자리 | 별자리 12궁 (생년월일에서 자동 산출 가능) |

### 분석 체계

- **만세력** — 생년월일시로 사주팔자를 세워 타고난 기질과 운의 흐름을 해석
- **MBTI 16유형** — 사고·행동 성향
- **혈액형 4유형** — 대중적 성격 분류
- **별자리 12궁** — 서양 점성 기반 성향

네 가지 체계를 **개별이 아닌 종합**으로 연결해 "온전한 나"의 프로필을 구성한다.

### 핵심 기능

1. **나에 대한 이해** — 네 가지 분석 체계를 종합한 '온전한 나' 프로필/리포트
2. **그날의 기운/성향 파악** — 매일 그날의 운기와 성향 조합을 바탕으로 한 데일리 가이드
3. **의사결정 도우미** — 이직·이별·선택 등 인생의 고비에서 성향과 운의 흐름을 종합해 조언
4. **대화형 AI 동반자** — 사용자의 프로필(사주·MBTI·혈액형·별자리) 맥락을 기억한 채 공감하며 대화하는 챗

## 핵심 가치 제안

데이터 기반의 분석을 **따뜻한 케어와 공감**으로 치환한다. 초기 전략은 '분석'보다 **'발견'과 '공감'** 에 초점 — "데이터를 통한 따뜻한 동행".

## 브랜딩 가이드

이 섹션의 톤·컬러·문체는 UI 카피, 마이크로카피, AI 응답 등 **모든 사용자 대면 텍스트에 일관되게 적용**합니다.

**톤앤매너 — '지적인 따뜻함':**
- 딱딱한 통계/분석 용어 금지. 마음을 어루만지는 공감형 문체 사용.
  - ❌ "당신의 사주 데이터 분석 결과입니다"
  - ✅ "당신의 오늘 기운은 ~하군요. 이럴 땐 이런 마음가짐이 도움이 될 거예요"

**컬러 팔레트:** 차가운 블루톤 지양. 베이지 / 차분한 코랄 / 딥 그린 조합으로 전문적이면서 포근하게.

**슬로건:**
- 메인: "모든 나를 잇다, 옴니마인드"
- 감성: "당신의 마음을 가장 잘 아는 모든 데이터"
- 강렬: "나보다 나를 더 잘 아는, 옴니마인드"

## 기술 스택 (확정)

- **Next.js** (TypeScript, App Router) — Vercel 무료 티어 배포
- **Supabase** — PostgreSQL DB + 카카오/구글 소셜 로그인
- **Gemini API 무료 티어** — LLM 해석 (Provider 인터페이스 뒤에 배치, 수익 발생 시 유료 모델로 교체)
- **운영 비용 원칙: 월 고정비 0원** — 무료 티어/무료 API만 사용

### 핵심 아키텍처 원칙

1. **계산과 해석의 분리** — 만세력·별자리 등 계산은 100% 규칙 기반 코드 (LLM 금지 영역), LLM은 문장 생성만
2. **3단 해석 엔진** — ①규칙 계산 → ②템플릿 문구 조합(항상 동작) → ③LLM 개인화(쿼터 있을 때만). LLM 없이도 서비스 완전 동작
3. **해석 캐싱** — 프로필 1회 생성, 데일리 하루 1회, 챗 하루 10회 제한. 같은 해석을 두 번 생성하지 않음
