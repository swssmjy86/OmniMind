# P5 AI 챗 '마음' Implementation Plan (v1.2)

> 설계서(SSOT): `2026-07-13-omnimind-design.md` §2(3단 해석·Provider), §6.4 · 로드맵 참조

**Goal:** 프로필 맥락을 기억하는 공감형 챗을 붙인다. **Gemini 무료티어**로 개인화하되, 쿼터 소진·오류·톤 위반 시 **템플릿 응답으로 무중단 폴백**. 하루 10회 제한(희소성 UX).

**Architecture:** 모든 해석 생성을 `InterpretProvider` 인터페이스 뒤에 둔다. 0단계=템플릿(항상 동작), 1단계=Gemini(쿼터 있을 때). 3단 폴백 체인이 실패를 흡수한다. 챗은 `chat_messages`에 저장(동반자의 기억). **API 키 없이도 템플릿 Provider로 완전 동작**(Method-A 방식).

```
사용자 메시지 → [InterpretProvider]
   ├─ GeminiProvider (키 있음 · 쿼터 남음) → 톤 가드 통과? → 응답
   └─ (실패/톤위반/쿼터소진/키없음) → TemplateProvider → 항상 응답
저장: chat_messages · 카운트: usage_counters(하루 10회)
```

## Global Constraints
- **LLM 장애가 정상 경로.** 쿼터 소진·키 없음도 오류 아님 — 템플릿으로 조용히 대체.
- Gemini 시스템 프롬프트 = §5.4 문체 규칙 전문. 출력은 톤 가드 재검사, 위반 시 폐기·폴백.
- 비밀키(`GEMINI_API_KEY`)는 `.env.local`·Vercel 환경변수, 커밋 금지. 서버에서만 호출.
- 월 고정비 0원 — Gemini Flash 무료 일일 쿼터. 하루 10회 제한이 쿼터 보호 + 프리미엄 판매 포인트.
