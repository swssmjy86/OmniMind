# P4 확산 장치 Implementation Plan (v1.1)

> 설계서(SSOT): `2026-07-13-omnimind-design.md` §6.2~6.6 · 로드맵: `2026-07-13-omnimind-roadmap.md`

**Goal:** 프로필/데일리를 **공유 카드**로 내보내 무료 유입 루프를 만들고, 아침 알림으로 재방문을 부르며, 비침습 광고와 성장 지표를 붙인다.

**Architecture:** 공유 카드는 `@vercel/og`(Edge, 무료)로 **서버에서 이미지 생성** — 프로필 컨텍스트를 쿼리로 받아 궁금증 유발형 카피를 그린다. 공유 UI는 Web Share API + 링크 복사, 링크엔 유입 추적 파라미터. 알림은 카카오 채널(수동 운영) 또는 PWA 웹푸시. 광고는 AdFit 슬롯.

## Global Constraints
- 월 고정비 0원 — `@vercel/og`(무료), AdFit/AdSense는 수익원. 카톡 채널 무료.
- 공유 카피는 §5.4 문체 + §6.2 "답을 다 주지 않는" 궁금증 유발형. 톤 가드 적용.
- 컬러·서체 P0 토큰. 카드는 베이지 바탕 + 명조 제목 + 오행/별자리 심볼.
- 개인정보: 카드에 생년월일시 노출 금지. 유형 조합(일간·오행·MBTI·별자리)만.
