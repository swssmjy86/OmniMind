# 옴니마인드 공식 채널 개설 가이드

- 작성일: 2026-07-16
- 대상: 공식 이메일, 카카오톡 채널, 인스타그램 — 실제 계정 개설은 담당자가 직접 진행
- 원칙: 월 고정비 0원(CLAUDE.md) — 세 채널 모두 무료 개설·운영 기준으로 작성

---

## 1. 공식 이메일

**추천: 무료 Gmail 계정으로 시작, 수익 발생 후 커스텀 도메인으로 전환.**
Google Workspace(예: hello@omnimind.kr)는 월 비용이 들어 "월 고정비 0원" 원칙과 맞지 않는다.
지금 단계는 무료 Gmail로 충분하고, 나중에 언제든 전환 가능하다.

### 개설 절차
1. gmail.com → "계정 만들기" → "비즈니스용 계정"
2. 아이디 후보(중복 확인 필요): `omnimind.official@gmail.com`, `hello.omnimind@gmail.com`, `omnimindteam@gmail.com`
3. 2단계 인증 설정(권장) — 카카오·인스타 등 다른 채널 계정 복구 이메일로도 쓰이므로 보안을 켜둔다
4. Gmail 설정 → 모든 설정 보기 → 일반 → 서명에 아래 서명 문구 등록

### 복사해서 쓰기 — 이메일 서명
```
온기를 담아,
옴니마인드 팀

모든 나를 잇다, 옴니마인드
https://omnimind.app  (배포 후 실제 도메인으로 교체)
```

## 2. 카카오톡 채널

설계서 §6.3(알림)에 "1순위 카카오톡 채널"로 이미 계획되어 있다. 완전 무료.

### 개설 절차
1. center-pf.kakao.com(카카오톡 채널 관리자센터) → 카카오 계정으로 로그인
2. "새 채널 만들기" 클릭
3. 채널 이름: `옴니마인드`
4. 검색용 아이디(영문, 중복 확인 필요): `omnimind`
5. 카테고리: 라이프스타일 > 운세/사주
6. 프로필 사진 등록, 아래 소개글 입력
7. 개설 후 "채널 홈 관리"에서 소개 문구·대표 이미지 추가 편집 가능

### 복사해서 쓰기 — 채널 소개글
```
사주, MBTI, 혈액형, 별자리…
흩어져 있던 당신의 조각들을 하나로 이어드려요 🌿

매일 아침, 당신만을 위한 오늘의 기운을 전해드릴게요.
— 모든 나를 잇다, 옴니마인드
```

### 향후 연동 메모
앱의 §6.3 알림 설계(아침 7~8시 1회, 하루 1회 초과 금지)와 이 채널을 나중에 카카오 알림톡/친구톡으로
연결할 수 있다. 채널 개설은 지금 해두고, 실제 발송 연동은 별도 기능 작업으로 진행.

## 3. 인스타그램

§6.2 공유 카드 확산 전략("MBTI 콘텐츠 문법 활용")과 직접 연결되는 채널.

### 개설 절차
1. 위에서 만든 공식 Gmail로 인스타그램 새 계정 생성
2. 사용자명 후보(중복 확인 필요): `omnimind.official`, `omnimind_kr`
3. 프로필 → "프로페셔널 계정으로 전환" → "크리에이터" 선택(무료, 인사이트 확인 가능)
4. 프로필 사진: 로고/워드마크 (아래 참고)
5. 소개글(bio) 등록, 링크에 서비스 URL 연결

### 복사해서 쓰기 — 인스타그램 bio
```
🌙 모든 나를 잇다, 옴니마인드
사주 × MBTI × 혈액형 × 별자리
오늘의 기운을 가장 잘 아는 곳
👇 나의 조각 이어보기
```

### 복사해서 쓰기 — 첫 게시물 문구 초안
```
MBTI만으로는 반쪽짜리예요.
사주까지 이어야 진짜 나.

1,920가지 조합 중, 당신은 어떤 이야기일까요?
```

앱의 `/api/card` 공유 카드(§P4-1)를 그대로 게시물 이미지로 쓸 수 있다 — 온보딩 완료 후 나오는
"나의 조각 카드"를 캡처해 첫 게시물로 올리는 방식을 추천.

### 프로필 이미지(로고) 메모
지금은 전담 디자이너 없이 시작하므로:
- 임시안: `/api/card` 카드의 일간 한자 워터마크(甲/乙/… + 오행 색) 모티프를 정사각형으로 잘라 프로필 사진으로 사용
- 정식 로고가 필요하면 Canva(무료 티어)로 브랜드 가이드(§5.2 컬러: 베이지·코랄·딥그린, §5.3 가는 라인 스타일·달/별/잎사귀 모티프)에 맞춰 제작 가능 — 필요하면 다음 작업으로 요청
